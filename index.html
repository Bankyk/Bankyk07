The basic language of the web:html

Posted by Bankole Khalid on Monday, of august 2024

All modern websites and web applications are built using fundamentaltechnologies: html, css, and javascript. These are the languages of the web.

In this post, lets focus on html. we will learn what html is all about, and why we should learn it

What is html?

html, stands for HyperTextMarkupLanguage that web developers use to structure and describe the concept of a webpage[not a programming language]

HTML, consists of elements that describe different types of contents: paragraphs, links, headings, images, video etc. Web browsers understand HTML and reader HTML code as websites.

In html, each element is made up of three parts:

  1. The opening tag
  2. The closing tag
  3. The actual element
